# metabList
public repo for the most recent version of metabList (for MARSS simulations). This is actively maintained and contains spin system specifications (i.e., chemical shifts, nuclei type, J-couplings, etc) for commonly measured in vivo 1H, 2H and 31P metabolites. If there's a missing metabolite, please raise an issue, ideally with a reference that defines the spin system. Access the latest version of MARSS here: inventions.techventures.columbia.edu/technologies/magnetic-resonance-spectrum--CU19215
